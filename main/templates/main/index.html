WELCOME

This is a website for demonstrating implementation of Huffman algorithm, Hamming(7, 4) code and OTP algorithm.

Instruction:

All you have to do is to pass through the 8 steps below starting from inputting some data. Each step will have a description and click to the "Next" button each time you complete a current step. Good luck!

START

STEP 1: INPUT

In the first step you should input some text. We will work with this input in next steps.

 

 


1 step

STEP 2: OTP ENCRYPTION

In the second step, we will encrypt our INPUT data with One-time pad. And for that you should input a KEY below and click to ENCRYPT button.

IMPORTANT! In OTP, the lengths of your input and key must be at least the SAME!

2 step

STEP 3: HUFFMAN ENCODING

In third step, we will encode our encrypted text from Step 2 with Huffman algorithm. Just click to the ENCODE button and you will get encoded string and compression ratio calculated by the formula: Number of bits before compression / Number of bits after compression




3 step

STEP 4: HUFFMAN DECODING

In the fourth step, we are going to decode the Huffman ENCODED string. Click to the DECODE button. The result should be same with result of Step 2.


4 step

STEP 5: HAMMING ENCODING

In this step, we will take the HUFFMAN ENCODED string from Step 3 and encode it with HAMMING(7, 4) CODE. So, it will add parity bits, just click to the ENCODE button. You will see it.


5 step

STEP 6: ADDING ERROR

Now, we will add some errors to our HAMMING ENCODED string from previous part. Click to the button ADD ERROR.


6 step

STEP 7: HAMMING DECODING

In this step, we will take string from previous step and DECODE it. But, firstly we should delete ERRORS that we added in Step 6. And secondly, delete parity bits from Step 5. Click to the DECODE button.


7 step

STEP 8: OTP DECRYPT

Finally, we will take the word from previous step and decrypt it. Just write the KEY and click to the DECRYPT button, you should get your initial text.

8 step

In order to start again, click to the "RESTART" button

RESTART
